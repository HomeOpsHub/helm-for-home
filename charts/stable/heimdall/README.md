## Description

Heimdall is an elegant application dashboard that lets you organize and access all your web applications and services from a single page. Originally deployed via [ansible-nas](https://github.com/davestephens/ansible-nas), this chart migrates Heimdall to a Kubernetes-native Helm deployment.

### Features

- Clean, customizable dashboard for all your self-hosted apps
- Enhanced application tiles with live stats (for supported apps)
- Search functionality across all configured applications
- Tag-based organization
- Persistent storage for configuration and custom icons
